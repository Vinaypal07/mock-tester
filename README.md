# MOCK MASTER
Conducting the mock Interviews

## MOCK INTERVIEWER 
Its made to conduct mock interview so that individuals can improve themselves. 
## ABOUT
-> the frontend provides you with multiple options when it comes to job roles so that you can select.
-> the interviewer asks you 20 questions. 
1) 15 questions are asked related to your job role. 
2) 5 questions are asked according to HR ROUND so questions are asked accordingly.
3) the difficulty of the questions ranges from
easy->medium->hard
4) after the session a status report is provided

PURPOSE: 
1)To conduct mock interviews to improve candidates confidence in interviews.
2)As well as to improve candidates knowledge. 
3)To remove the fear of interviews from the candidate so that they can perform well.
4)By using this web application you can analyse your weak points and improve. 
5)In fact candidates can keep track of their time so that they can improve their time management skills during problem solving.
